## 摘要
在本文中，我们提出了一种概念验证型的图神经网络（GNN）模型，该模型通过对图结构和连接特征进行精确建模，能够成功预测网络流级流量（NetFlow）。我们使用滑动窗口将网络流量划分为等尺寸的异构双向图，其中包含 IP、端口（Port）和连接（Connection）节点。随后，我们利用 GNN 对图结构和连接特征的演化进行建模。我们的方法在识别连接所属的端口和 IP 方面表现出优越的结果，同时在特征重建方面与强大的预测基准模型相比仍具竞争力。总之，我们的工作展示了 GNN 在逐流（per-flow）NetFlow 预测中的应用。

## Abstract
In this paper, we propose a proof-of-concept Graph Neural Network model that can successfully predict network flow-level traffic (NetFlow) by accurately modelling the graph structure and the connection features. We use sliding-windows to split the network traffic in equal-sized heterogeneous bidirectional graphs containing IP, Port, and Connection nodes. We then use the GNN to model the evolution of the graph structure and the connection features. Our approach shows superior results when identifying the Port and IP to which connections attach, while feature reconstruction remains competitive with strong forecasting baselines. Overall, our work showcases the use of GNNs for per-flow NetFlow prediction.

### 1. 核心问题与研究动机
*   **核心问题**：如何在细粒度（逐流，Per-flow）级别上准确预测网络流量（NetFlow）。
*   **研究背景**：传统的网络流量预测多关注聚合指标（如链路利用率），虽然简单但会掩盖“微突发”流量等重要细节。逐流预测由于数据量大、维度高（数值与类别混合）且具有复杂的关联性，一直被认为极具挑战性。
*   **动机**：网络流本质上不是孤立的，而是更大通信模式的一部分。图神经网络（GNN）能够捕捉这种关系结构，因此比传统的表格化建模或单纯的时间序列模型更适合处理具有拓扑属性的 NetFlow 数据。

### 2. 方法论
*   **核心思想**：将网络流量建模为异构图，并利用**预测型掩码图自编码器（Predictive Masked Graph Autoencoder）**，通过当前时间窗口的图结构预测下一个时间窗口的图结构和流特征。
*   **关键技术细节**：
    *   **图构建**：使用滑动窗口（长度 512）将流量切分为异构双向图。节点类型包括：IP 节点、端口节点（基于 OHE 映射的 8 类）和连接节点（包含流特征）。
    *   **模型架构**：基于 GraphSAGE 算子构建图自编码器（GAE）。
    *   **掩码机制**：借鉴 GraphMAE 和 HGMAE，对节点特征进行掩码，并采用随机掉边（Edge Dropping）策略强制模型学习非平凡的结构信息。
    *   **解码器**：
        *   **特征解码**：使用简单的多层感知机（MLP）重建掩码后的节点特征。
        *   **结构解码**：使用增强型点积（Enhanced Dot Product）结合可学习权重来预测未来的图连接关系。
    *   **损失函数**：采用复合损失函数 $L_{Total} = \alpha \cdot StructLoss + (1 - \alpha) \cdot FeatLoss$，平衡结构重建与特征预测。
